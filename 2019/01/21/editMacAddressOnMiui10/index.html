在miui10上修改wifi的mac地址

2019-01-21 手机, 技巧 0 评论

本人手机1+3t, 买的时候不知道在哪看的说是”刷机不费力…刷坏还管修…”

虽然是奔着这个目的去的,但是到手2年多 也没刷过几次. 前几天闲的蛋疼想体验回小米的系统.
果然 爽歪歪,时隔4年打开还算熟悉的界面, 相册,联系人,短信 aha 一个都没丢.

但是,在公司网络 wifi 总是时断时连,找了半天原来是:刷的这个H大版本的 miui10 for 1+3t 系统 wifi 的 mac 地址大家都一样…(克隆出来的?)网络中产生了冲突

那么让我们看看怎么解决.

  • 下载 root explorer
  • 使用 root explorer 赋权并打开/修改手机文件 /system/etc/firmware/wlan/qca_cal/wlan_mac.bin
  • 我这里原来是空白的,所以直接添加 Intf0MacAddress=000AF58989CC (等号后面是你要换成的 mac 地址,没有冒号)

关掉 wifi 重连一下试试, aha~ 网络冲突没有了, 终于可以安心的连公司 wifi 上网了.
